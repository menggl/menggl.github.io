服务启动异常整理

发表于 2018-10-29 | 分类于 整理

tomcat7-maven-plugin 启动tomcat时报错

错误信息如下：

Caused by: java.lang.LinkageError: loader constraint violation: loader (instance of org/apache/catalina/loader/WebappClassLoader) previously initiated loading for a different type with name “javax/servlet/ServletContext”

主要错误信息在这里
Caused by: java.lang.LinkageError: loader constraint violation: loader (instance of org/apache/catalina/loader/WebappClassLoader) previously initiated loading for a different type with name "javax/servlet/ServletContext"
说明jar包冲突，但是我们明明在pom.xml中配置的信息是

这样的啊，已经将scope设置成了provided模式了，为啥还会有servlet冲突问题啊

接着我们将这个配置注释掉，使用mvn dependency:tree 查看还有哪些servlet jar包引用

最后我们发现

  • 在com.alibaba:dubbo:jar:2.8.4h:compile包中有一个javax.servlet:javax.servlet-api:jar:3.1.0:compile
  • 在org.apache.cxf:cxf-rt-transports-http:jar:2.5.2:compile包中有一个org.apache.geronimo.specs:geronimo-servlet_2.5_spec:jar:1.1.2:compile
    都使用exclusion排除掉，问题解决
