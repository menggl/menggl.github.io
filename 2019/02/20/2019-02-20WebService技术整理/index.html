wsdl:binding节点

根据wsdl:service节点中的binding配置在wsdl:binding节点中找到对应的节点

根据wsdl:binding节点中的type值找到对应的wsdl:portType节点

总结一下：

  1. 是整个webservice的服务视图，它包括了所有的服务端点。
  2. 为每个端口定义消息格式和协议细节。
  3. 描述webservice可被执行的操作，以及相关的消息，通过binding指向portType。
  4. 定义一个操作（方法）的数据参数（可有多个参数）。
  5. 定义webservice使用的全部数据类型。

参考文档：https://blog.csdn.net/u012702547/article/details/45667623
